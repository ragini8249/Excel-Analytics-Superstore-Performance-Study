# Excel-Analytics-Superstore-Performance-Study
#Business Analytics Project Using Excel — Sales, Profit &amp; Operational Insights

This project demonstrates complete Business Analytics (BA) using Microsoft Excel. The dataset contains business transactions such as orders, returns, shipping details, customers, sales, and profit information. The goal of this project is to analyze business performance, identify trends, locate problem areas, and generate meaningful insights for decision-making.

# Dataset Overview

The Excel file contains the following sheets:

Orders — primary dataset with 21 columns
(Order Date, Ship Date, Customer Details, Segment, Region, Product Category, Sales, Quantity, Profit, Discount, etc.)

Returns — records of returned orders

People — regional managers

Additional sheets for supporting calculations/analysis

# Objectives of the Project

This project focuses on discovering key business insights, including:

Sales & Profit trends

Customer & Regional performance

High-selling categories & sub-categories

Discount impact on profit

Returned orders impact

Identifying loss-making products

Operational metrics like Shipping Modes & Delivery Time

# Tools & Techniques Used

Excel Functions: VLOOKUP/XLOOKUP, IF, SUMIFS, COUNTIFS, DATE formulas

Pivot Tables: Sales, Profit, Category analysis, Customer segmentation

Pivot Charts: Trend analysis visualizations

Data Cleaning: Removing duplicates, formatting dates, correcting data types

Dashboards: Interactive Excel dashboard to summarize KPIs (optional if created)


#1️ Data Understanding

The Orders sheet contains 21 columns, including:

Customer Information

Geographic Information

Sales & Profit metrics

Product Categories

Shipping details

This is a typical retail superstore dataset used for business analysis.

#2️ Data Cleaning

Performed the following steps:

Converted Order Date / Ship Date into proper date format

Removed duplicate orders

Checked missing values

Standardized Region, Category, and Segment labels

Calculated Delivery Days = Ship Date – Order Date

#3️ Exploratory Data Analysis (EDA)

Using Excel Pivot Tables & formulas, the following insights were derived:

✔ Sales & Profit by Category

Compared Furniture, Office Supplies, Technology segment performance

Identified the most profitable and least profitable categories

✔ Trend Analysis

Monthly & yearly Sales trend

Seasonal peaks and dips

Profit trend compared against Sales

✔ Region & Segment Analysis

Region-wise revenue contribution

Consumer vs Corporate vs Home Office performance

✔ Discount vs Profit Relationship

Analyzed how higher discounts reduced profitability

Identified categories where discount strategy needs revision

✔ Returned Orders Impact

Connected Returns sheet with Orders using Order ID

Measured financial loss due to returned orders

✔ High Loss Products

Identified products where profit is negative despite high sales

Recommended discontinuation or pricing strategy changes
